# 功能测试指南

安装完成后，请按以下步骤测试插件功能：

## ✅ 基础功能测试

### 1. 日历界面
- [ ] 插件图标显示在工具栏
- [ ] 点击图标弹出日历窗口
- [ ] 显示当前月份的日历
- [ ] 今日日期有特殊标识（边框高亮）
- [ ] 点击左右箭头可切换月份
- [ ] 点击"今天"按钮返回当前月份

### 2. 日期选择
- [ ] 点击日历中的任意日期
- [ ] 选中的日期有紫色边框标识
- [ ] 下方显示选中日期的中文格式
- [ ] 可以连续点击不同日期切换

### 3. 添加待办事项
- [ ] 选择任意日期
- [ ] 点击"+ 添加"按钮
- [ ] 弹出输入框
- [ ] 输入待办事项（如：完成工作汇报）
- [ ] 点击确定后事项出现在列表中
- [ ] 刷新页面后事项依然存在

### 4. 待办事项状态切换
- [ ] 点击待办事项前的复选框
- [ ] 事项标题显示删除线（表示已完成）
- [ ] 再次点击可恢复未完成状态
- [ ] 已完成事项颜色变淡

### 5. 编辑待办事项
- [ ] 点击待办事项项的"编辑"按钮
- [ ] 弹出编辑框，当前内容已填入
- [ ] 修改内容并确认
- [ ] 事项标题更新

### 6. 删除待办事项
- [ ] 点击待办事项的"删除"按钮
- [ ] 弹出确认对话框
- [ ] 点击确定后事项被删除
- [ ] 事项从列表中消失

### 7. 日历背景色变化
- [ ] 为某一天添加1-2个待办事项，该日期显示浅绿色背景
- [ ] 添加3-4个待办事项，显示浅橙色背景
- [ ] 添加5-6个待办事项，显示浅红色背景
- [ ] 添加7+个待办事项，显示深红色背景
- [ ] 没有待办事项的日期显示白色背景

## 🔍 高级功能测试

### 8. 搜索功能
- [ ] 在顶部搜索框输入关键词
- [ ] 实时筛选匹配的待办事项
- [ ] 清空搜索框恢复显示全部事项
- [ ] 搜索支持标题和描述匹配

### 9. 事件列表
- [ ] 选择有多个待办事项的日期
- [ ] 查看该日期的所有待办事项列表
- [ ] 事项按创建时间排序显示
- [ ] 区分显示待办和已完成事项

### 10. 数据持久化
- [ ] 添加几个待办事项
- [ ] 关闭插件弹出窗口
- [ ] 重新打开插件
- [ ] 数据完整保留

### 11. 月份切换
- [ ] 点击"上个月"按钮切换到上个月
- [ ] 点击"下个月"按钮切换到下个月
- [ ] 切换月份后，之前添加的待办事项依然存在
- [ ] 不同月份的日期显示不同的背景色

## 🤖 AI功能测试（需要API密钥）

> **注意**：AI功能需要硅基流动API密钥，可在 https://cloud.siliconflow.cn/ 免费注册

### 12. 配置API密钥
- [ ] 点击右上角⚙设置按钮
- [ ] 在设置中输入API密钥
- [ ] 保存设置
- [ ] 设置持久化保存

### 13. AI总结生成
- [ ] 选择有2-3个待办事项的日期
- [ ] 点击"AI总结"按钮
- [ ] 显示"正在生成总结..."
- [ ] 等待几秒后生成智能总结
- [ ] 总结内容包含工作总结、效率分析、明日建议
- [ ] 点击×关闭总结窗口

### 14. AI总结内容质量
- [ ] 总结准确列出当日已完成和未完成事项
- [ ] 提供合理的效率分析
- [ ] 给出实用的明日建议
- [ ] 总结内容为中文
